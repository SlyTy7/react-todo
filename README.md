# My To-Do

A basic to-do list built with ReactJS.  You can add new to-dos, edit current to-dos, and delete completed and uncompleted tasks.

[Check it out here!](https://slyty7.github.io/react-todo/)

## Built With

  * HTML
  * CSS
  * JavaScript
  * ReactJS
  * Node
  * NPM
  * Material UI
